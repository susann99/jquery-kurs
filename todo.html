1. Was ist jQuery

Quelle: Frank Bongers, Maximilian Vollendorf, jQuery - Das Praxisbuch, 2010, Galileo Press
Ein Javascript-Framwork:
  • Erweiterungen einer Programmiersprache wie eine Art Akkuschrauber mit schönen Aufsätzen (Toolset)
  • Weiter JS-Frameworks:
    • Ext JS - http://www.extjs.com/ + http://www.sencha.com/
    • Dojo - Dojo Community http://www.dojotoolkit.org/ + node.js
    • GWT - http://code.google.com/webtoolkit/ (typisch Google, immer etwas komplex)
    • script.aculo.us - Frackles
  • Lib oder Framework? Bei Javascript bis vor kurzem Browserabhängig
  • Codeknappheit $(): $("div#d1").show();
  • Verkettbarkeit (Chaining): $("div").hide().css(color,"blue").slideDown();
  • Erweiterbar (Plugins): Oberflächenbibliothek jQuery UI
